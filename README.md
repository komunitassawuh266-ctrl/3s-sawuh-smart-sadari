# 3s-sawuh-smart-sadari
SAWUH SMART SADARI adalah inovasi digital untuk mendukung perempuan dalam melakukan pemeriksaan payudara sendiri secara rutin dan terstruktur.  Dengan panduan langkah SADARI yang lengkap, monitoring bulanan, dan edukasi tanda bahaya, program ini hadir sebagai upaya deteksi dini kanker payudara demi meningkatkan peluang kesembuhan.
